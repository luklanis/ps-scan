In this mode ESR Scanner works like a payment slip reader (ex. paypen). Simply scan your payment and the code row will be sent (over wifi) to your computer.

How to use
  1. Download and extract ESRReceiver (you need Java) from http://www.wuala.com/gulb/open/ESRReceiver/ESRReceiver.zip to your PC
  2. Enable wifi, bluetooth tether or usb tether on your mobile
  3. Start ESRScanner and go to Settings
  4. Check "Enable slip-reader mode" and go back
  5. Fill in the showed (on the bottom right corner) ip address on ESRReceiver and press "Connect"
  6. Wait until ESR Receiver becomes connected state
  7. Place the cursor on your computer where the code row should be placed
  8. Scan your code

If you have slip-reader mode enabled, you can't go to a detail of a payment. Instead you can re-send the code row if you select it in the history view.
