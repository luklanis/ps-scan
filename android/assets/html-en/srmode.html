In this mode ESR Scanner works like a payment slip reader (ex. paypen). Simply scan your payment and the code row will be sent (over wifi) to your computer.

How to use
  1. Download ESRReceiver (you need Java) from http://www.wuala.com/gulb/open/ESRReceiver
  2. Enable wifi, bluetooth tether or usb tether on your mobile
  3. Press to activate payment slip reader mode
  4. Fill in the showed (on the bottom right corner) ip address on ESRReceiver and press "Connect"
  5. Wait until ESRReceiver becomes connected state
  6. Set the cursor on your computer to the location where the code row should be placed
  7. Scan your code

If you have slip-reader mode activated, the code row wouldn't be saved on the phone.
