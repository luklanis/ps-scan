In diesem Modus arbeitet der ESR Scanner wie ein Belegleser (z.B. paypen). Einfach den Einzahlungsschein einscannen und schon wird es (über WLAN) an deinen Computer gesendet.

Anleitung
  1. Downloade und extrahiere den ESRReceiver (Java wird auch benötigt) von https://github.com/luklanis/esr-receiver/downloads auf deinem Computer
  2. Starte das WLAN, Bluetooth Tether oder USB Tether auf deinem Smartphone
  3. Starte den ESRScanner und gehe zu den Einstellungen
  4. Selektiere "Beleglesermodus einschalten" und gehe zurück
  5. Trage die IP Adresse, die jetzt im ESRScanner unten links angezeigt wird, im ESRReceiver ein und drücke "Connect"
  6. Warte bis der ESR Receiver den Status auf "Connected" setzt
  7. Plaziere den Cursor in das Feld, wo die Kodierzeile eingetragen werden soll.
  8. Scannne die Zahlung

Bei eingeschaltenem Beleglesermodus werden keine Zahlungdetails angezeigt. Wenn man im Verlauf eine Zahlung selektiert, wird diese noch mal gesendet.
