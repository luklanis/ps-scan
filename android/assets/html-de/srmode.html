In diesem Modus arbeitet der ESR Scanner wie ein Belegleser (z.B. paypen). Einfach den Einzahlungsschein einscannen und schon wird es (über WLAN) an deinen Computer gesendet.

Anleitung
  1. Downloade den ESRReceiver (es wird Java benötigt) von http://www.wuala.com/gulb/open/ESRReceiver
  2. Starte das WLAN, Bluetooth Tether oder USB Tether auf deinem Smartphone
  3. Drücke um den Beleglesermodus zu aktivieren
  4. Trage die IP Adresse, die jetzt im ESRScanner unten rechts angezeigt wird, im ESRReceiver ein und drücke "Connect"
  5. Warte bis der ESRReceiver den Status auf "Connected" setzt
  6. Plaziere den Cursor in das Feld, wo die Kodierzeile eingetragen werden soll.
  7. Scannne die Zahlung

Bei aktiviertem Beleglesermodus wird die gescannte Codierzeile nicht gespeichert.

Um Codierzeilen von bereits gescannten Zahlungen zu senden, muss man den Beleglesermodus deaktivieren, in den Verlauf gehen und dort drücken.
